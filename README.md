## 참고 사항

- 빌드 산출물(`.vs/`, `x64/`, `*.pdb` 등)은 `.gitignore`로 제외되어 있습니다.
- HW3의 3D 씬 에셋(`Scene/`, `Data/` — Bistro 모델·텍스처)은 용량이 커(개별 파일 100MB 초과) 저장소에서 제외했습니다. 실행하려면 수업에서 제공된 에셋을 해당 경로에 배치해야 합니다.
